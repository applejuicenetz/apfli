Installation:

make
make install (als root)

________________________________________________________________________________

Falls es Probleme beim ausfuehren gibt koennen folgende Dinge helfen:

1.)Nachsehen ob das Verzeichnis ~/.apfli existiert. Wenn nicht dann erstellen:
mkdir ~/.apfli
    
Es kann sein das von der vorherigen ApfLi Version noch die Datei ~./apfli
existiert. Diese MUSS geloescht werden!

2.)Die Datei ~/.apfli/settings loeschen.

3.)http://apfli.sourceforge.net besuchen und in die FAQ sehen.

4.)Wenn das Problem immernoch nicht geloest ist, auf www.applejuicenet.de gehen
und die Programmierer wuest beschimpfen :) (nochbesser natuerlich freundlich
nachfragen)
